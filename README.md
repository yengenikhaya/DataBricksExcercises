# DataBricksExcercises

Sales Analysis Methodology – Bright Coffee Shop
1. Objective
To analyse point of sale transaction data in order to:
•	Identify key sales drivers
•	Understand customer behaviour by time slot and location
•	Generate actionable, revenue focused recommendations for leadership
2. Data Scope
•	Time period: January–July 2023
•	Locations: Lower Manhattan, Hell’s Kitchen, Astoria
•	Transactions: Thousands of individual sales records
•	Customer segment analysed: Low Spender segment (dominant in dataset)
•	Time segmentation: 
o	Rush Hour
o	Mid Morning
o	Afternoon
o	Evening
o	Night
3. Data Preparation
•	Cleaned and standardised product names and categories
•	Grouped products into: 
o	Coffee
o	Tea
o	Bakery
o	Drinking Chocolate
o	Add ons (syrups, flavours)
•	Normalised pricing for accurate comparison
•	Categorised transactions by location, time slot, and product type
4. Analytical Techniques Used
•	Descriptive analysis: Volume, frequency, average spend
•	Time based analysis: Sales patterns by daypart
•	Location analysis: Product performance per store
•	Product mix analysis: Best selling vs underperforming items
•	Basket behaviour: Single item vs multi item purchases
5. Visualisation Approach
The following visuals were prepared for executive clarity:
•	Bar charts (Top selling products)
•	Heatmaps (Sales by time slot and location)
•	Pie charts (Product category mix)
•	Trend views (Peak vs slow periods)
6. Key Limitations
•	Majority of data represents low spender customers
•	Limited demographic data
•	Insights strongest for operational and marketing optimisation, not pricing elasticity modelling
